# 画册设计原则

## 核心理念

> "一张照片在手机屏幕上看，和真正变成一本书来翻阅，体验大不相同。把记忆从数据流里打捞出来，捧在手心里。"

### 你是唯一主体

- AI 不替你表达，只帮你排版
- 文字引导而非文字生成
- 照片是主体，文字是辅助
- 不编造体验，不添加幻觉

### 设计哲学（来自阿兰·德波顿《旅行的艺术》）

- 旅行的意义不在于目的地，而在于途中的观察
- 摄影是一种转译——把你看到的记录下来，让别人解读
- 好的画册不是"好看的照片集合"，而是一个有节奏的叙事

---

## 排版技巧（来自狂阿弥画册经验）

### 1. 先想象，再动手
第一步不是选照片，而是想象你要做一本什么样的书。
关键词定调：回忆录式的？青春的？轻盈的？私人的？

### 2. 选图：给后期留余地
- 600 张 → 选 70 张 → 实际用约 60 张
- 多选几张做排版时的冗余
- 60 页的线装本太厚合不上（"像炸开的三明治"）

### 3. 排序比排版更重要
照片不是随便塞进去的，要有逻辑线：
- 时间线：出发 → 途中 → 到达 → 回程
- 情绪线：压抑 → 释放 → 高潮 → 平静
- 空间线：城市 → 乡村 → 海边 → 山上

### 4. 排版小技巧

**并置**：左右页放两张不相关的照片，产生新意义
```
[一双旧鞋] | [一片新海]
```

**旋转**：竖构图变横放，打破方向感

**前后呼应**：前页远景 → 后页特写
```
[远景：海滩上一个人] → [特写：脚踩在沙子里]
```

**相似元素串联**：圆形 → 圆形，自然过渡
```
[圆形：咖啡杯] → [圆形：月亮]
```

**大小节奏**：照片一大一小交替
```
[大图] → [小图+文字] → [大图] → [网格小图]
```

## 字体规则

| 用途 | 字体 | 说明 |
|---|---|---|
| 标题 | Noto Serif SC | 衬线，视觉重音 |
| 正文 | Noto Sans SC | 非衬线，信息密度 |
| 元数据 | IBM Plex Mono | 等宽，装饰节奏 |
| 英文 | Playfair Display | 衬线，杂志感 |

**核心规则**：
- 衬线用于标题、金句、数字——"视觉重音"
- 非衬线用于正文、说明——"信息密度"
- 等宽用于日期、标签、页码——"装饰节奏"

## 色彩使用

- 暖墨主题：适合有温度的照片（人物、旅行、日常）
- 银盐主题：适合冷静的照片（街拍、建筑、黑白）
- 照片本身就是色彩，背景色不要抢
- dark 页面（深底）用于：封面、章节分隔、引言、高潮
- light 页面（浅底）用于：正文、网格、说明

---

## 节奏控制

```
封面（暗）→ 引言（暗）→ 章节（暗）→ 照片（亮）→ 叙述（亮）
→ 网格（亮）→ 全屏（暗）→ 章节（暗）→ 照片（亮）→ 结尾（亮）
```

- 每 3-4 页换一次明暗
- 连续不超过 3 页同色
- 高潮照片用全屏暗底，仪式感最强
- 网格照片用亮底，信息密度最高

---

## 打印建议

- A4 横向：适合展示大图
- A5 纵向：适合手持翻阅
- 方形（20x20cm）：适合 Instagram 风格
- 线装本：最多 60 页，太厚合不上
- 铜版纸：照片色彩还原最好
- 哑光纸：手感好，不反光
