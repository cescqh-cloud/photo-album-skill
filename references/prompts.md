# 提示词配置

所有可调整的提示词集中在此文件。修改后即刻生效。

---

## 模式选择提示

当用户没有指定模式时，使用以下引导：

```
你想怎么整理这些照片？
1. 杂志模式 —— 你告诉我主题和时间，我帮你排版
2. AI 全权 —— 我自动分类处理所有照片
3. 预设类型 —— 选一个场景（出去玩 / 记录人物 / 街拍 / 活动探店）
```

---

## 图片分析提示

分析每张照片时使用：

```
分析这张照片，输出以下信息（JSON 格式）：
- description: 一句话描述画面内容
- mood: 情绪基调（温暖/安静/活力/压抑/自由/孤独/欢乐/怀旧）
- composition: 构图特点（特写/远景/中景/对称/留白/俯拍/仰拍）
- color_tone: 色调（暖/冷/中性/高饱和/低饱和）
- subjects: 主要内容标签（人物/风景/建筑/食物/动物/植物/天空/水面/街道/室内）
- quality_estimate: 粗略质量评估（好/一般/差）
- suggested_layout: 建议的布局类型（full-bleed/duo/grid/text-photo/quote）
```

## 选图提示

从候选照片中筛选时：

```
你有 {total} 张照片，目标选出约 {target} 张。
选图原则：
1. 去掉模糊、过曝、重复的照片
2. 保留有叙事价值的照片（有情绪、有故事、有转折）
3. 大图、特写、远景各保留一定比例
4. 给排版留冗余（多选几张，排版时再精简）
5. 开场和收束的照片优先保留
```

## 排序提示

确定叙事线时：

```
基于选出的照片，确定一条叙事线。选项：
1. 时间线：按拍摄时间排列
2. 空间线：按地点/场景转换排列
3. 情绪线：从低沉到高昂（或反过来）
4. 主题线：按内容主题分组

选择最适合这组照片的叙事线，并说明理由。
```

## 文字引导提示

当用户不知道写什么时，逐个提问（不要一次全问）：

### 通用引导
```
你不需要写很多，几句话就够了。我问你几个问题，你随便回答就行：

"拍这张照片的时候，你在想什么？"
```

### 按下快门的心情
```
- "是什么让你停下来举起相机？"
- "这张照片里，最吸引你的是什么？"
- "如果用一个词形容当时的感受，是什么？"
```

### 最近发生的事情
```
- "拍这组照片前后，生活中有什么变化？"
- "这段时间，你最常去的地方是哪里？"
- "有没有什么小事，让你印象深刻？"
```

### 场景专用引导

**出去玩**：
```
- "这趟旅程的起点是什么？"
- "路上最意外的发现是什么？"
- "如果用一个颜色形容这趟旅行，是什么颜色？"
- "有没有一个瞬间，你觉得'值得来这一趟'？"
```

**记录人物**：
```
- "你最想记住这个人的什么？"
- "有没有一个瞬间，你觉得'就是这个'？"
- "这张照片背后的故事是什么？"
- "如果十年后看这张照片，你希望记住什么？"
```

**街拍**：
```
- "你走在路上，是什么让你按下快门？"
- "这座城市给你什么感觉？"
- "你希望别人从这张照片里看到什么？"
- "有没有一个细节，只有你注意到了？"
```

**活动/探店**：
```
- "这个活动/地方最打动你的是什么？"
- "有没有一个细节，别人可能注意不到？"
- "你会怎么向朋友描述这个地方？"
- "如果再来一次，你会拍什么不一样的？"
```

## 页面标题生成提示

为每个章节/页面生成标题时：

```
基于这组照片的内容和用户的文字素材，生成：
1. 画册标题（4-8 个字，简洁有力）
2. 副标题（一句话，补充说明）
3. 章节标题（如果有多组照片，每组一个 2-4 字标题）
4. 页面 kicker（每页上方的小引导句，5-10 字，有节奏感）

要求：
- 不要用"美丽的""难忘的""精彩的"等空洞形容词
- 用具体的事物或动作
- 像朋友之间的对话，不要像作文
```

## 尾页文字提示

生成尾页信息时：

```
基于用户提供的素材，生成尾页内容：
1. 一句话总结（不超过 20 字）
2. 时间标注
3. 可选：一句话留给未来翻阅的自己

风格：克制、温暖、不煽情。
```

## AI 评分提示（可选功能）

对照片进行简单评分：

```
对这张照片进行评分（1-10），参考以下维度：
- 构图：是否有视觉重心？是否遵循三分法或对称？
- 光线：是否自然？是否有有趣的光影？
- 情绪：是否传达了某种感受？
- 瞬间：是否捕捉到了某个特别的时刻？

输出格式：
- score: 总分（1-10）
- strength: 最大的优点
- suggestion: 如果要改进，建议是什么

注意：评分仅供参考，不是竞赛。普通人的真实记录比技术完美的照片更有价值。
```

## 排版决策提示

决定每张照片用什么布局时：

```
基于照片内容和叙事位置，决定布局：
- 全屏（full-bleed）：情绪高潮、转折点、开场/收束
- 双页并置（duo）：对比、呼应、相似元素
- 网格（grid）：同主题多图、细节展示、碎片记录
- 左文右图（text-photo）：有叙述价值的照片
- 照片+说明（caption）：有明确故事的照片
- 引言（quote）：用户的心情记录、金句

节奏原则：
- 大图 → 小图 → 文字 → 大图
- 每 3-4 页换一次明暗
- 不要连续出现两个相同布局
```
